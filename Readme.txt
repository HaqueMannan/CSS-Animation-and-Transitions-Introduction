Introdction to CSS Animations & Transitions
====================================================================

We will learn two principals for CSS Animations & Transitions:
1. CSS Transitions
2. CSS Animation & Keyframes

We will also learn the principal of what, how and why you should animate certain elements and which elements you should not.

-----------------------------------------------------------------------

Animation (definition)
- Instill with life
- Having the appearance of movement

CSS Movement (Meathods)
- Transition Property
- Animation Propery & Keyframes

-----------------------------------------------------------------------

This tutorial will be split into two parts:
Part 1 will demonstrate Transitions and Syntax while Part 2 will demonstrate the Animation & Keyframes and Syntax.
Additional Animation files will be added for demonstration purposes. This will build a short introduction to CSS animation properties.
